# App hecha con el framework de Vue 3 + Vite

Bienvenidos a la presentacion conceptual #2 de mi proyecto de practica profesional
Por Guillermo Arturo Lezcano Gonzalez

En que consiste la app?
Es una aplicacion de gestion de tickets y documentos centralizada para el departamento interno de la DGA de la UP

Que buscamos resolver?
Agilizar proceso de consultas en recepcion y busqueda de documentos internos dentro del departamento.

Alcance del proyecto
WIP: Alcance para pruebas internas dentro del departamento de la DGA.
DMS: Implementacion completa en un srvidor internos para funcionamiento optimo en multiplataforma.
